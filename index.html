University of Kentucky

By: Marcus Rodriguez

About

    The University of Kentucky is a public land-grant research university in Lexington, Kentucky. It was founded in 1865 and has since then grown into a large campus that is now the institution with the highest enrollment rate in the state.

Data Used in this Project

    Data points were collected through the use of GPS features on an iPhone XS and the map was construced using Mapbox Studio.

Tools Used in this Project

  • Data is shown on Mapbox web map with geolocation services enabled. And constructed the map from scratch by adding in three text layers and several custom tilesets.
  • QGIS 3.2 to convert text file into point locations to be used in datasets within Mapbox.

Project Assets

  • Featured points as GeoJSON file
